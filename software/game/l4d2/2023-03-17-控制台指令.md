# 指令

## 基础

### 队友传送

```sh
Warp_all_survivors_here
```

### 开启作弊模式

```sh
Sv_Cheats 1
```

### 无敌

```sh
god 1
```

### 穿墙

```sh
noclip
```

### 设置第三人称

```sh
thirdpersonshoulder 1
```

- 绑定到按键 <kbd> g </kbd>

```sh
bind g "thirdpersonshoulder 1"
```

## 武器

### 属性

#### 无限弹药不换弹夹

```sh
sv_infinite_ammo 1
```

#### 激光

```sh
upgrade_add laser_sight
```

#### 榴弹

```sh
give weapon_grenade_launcher
```

### 种类

#### m60

```sh
give rifle_m60
```

## 地图

```sh
map [地图代码]
```

### 突变模式

```sh
map [地图代码] [模式名]
```

#### 猎头者

```sh
map [地图代码] mutation2
```
