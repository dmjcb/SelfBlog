# 压缩视频

```sh
ffmpeg -i "source.mp4" -threads 16 -preset ultrafast -c:v libx265 -crf 28 "target.mp4"
```

## 硬件加速

```sh
ffmpeg -hwaccel cuvid -hwaccel_output_format cuda -i "source.mp4"  -c:v hevc_nvenc -map_metadata 0 -preset slow -cq 33 "target.mp4"
```
