# 结构体

C语言中结构体是一种数据类型, 存放一组不同类型的数据

## 定义

```c++
struct Stu {
    char *name;
    char *num;
    int age;
};
```

## 构造函数

- 无参构造函数

```c
Stu(){}
```

- 含参构造函数

```c
Stu(char *name, char *num, int age){
    this->name = name;
    this->num = num;
    this->age = age;
}
```

## 结构体数组

```c++
Stu s[3];

s[0] = Stu("WANG", "A-234", 19);
s[1] = Stu("Han", "A-235", 19);
s[2] = Stu("Liu", "A-236", 19);

for(int i = 0; i < 3; i++){
    pritf("%s %s %d\n", s[i].name, s[i].num, s[i].age)
}
```
