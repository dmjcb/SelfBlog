# 数据类型

## 整形

### 无符号类型

#### unsigned

- unsigned int

无符号整型, 长32位

```c
unsigned int

uint32_t

// sizeof(uint32_t) = 32
```

- unsigned long long

```c
unsigned long long

unsigned long long int

uint64_t

// sizeof(uint64_t) = 64
```

- unsigned char

```c
unsigned char

uint8_t

// sizeof(uint8_t) = 8
```

## 浮点型

### float

大小为4字节, 32位

### double

大小为8字节, 64位


- 判断相等

```c
int a;
double b;

// 向下取整
printf("%.0lf\n", floor(a * b * 0.1));
```

- 相除余数

```c
double a, b;

scanf("%lf%lf", &a, &b);

int k = a / b;
double r = a - k * b;

printf("%g", r);
```
