# 弹窗

## 提示框回调

```js
alert()
```

## 确认提示框

```js
const msg = "内容？\n\n请确认！"

if (confirm(msg)){
    // ...
} else {
    // ...
}
```

## 输入提示框

```js
const value = prompt("Input Text", "Default Text")

if (value != null && value != "") {
    alert(value)
}
```
