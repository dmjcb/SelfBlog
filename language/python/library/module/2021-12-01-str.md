## 定义

```py
s = "Hello World"
```

### input

Python input() 接受输入数据均为字符串格式

```py
s = input()

# Helle World
print(s)

# <class 'str'>
print(type(s))
```

## 操作

### 切片

| 语法          | 范围          |
| ------------- | ------------- |
| $[start:end]$ | [start, end)  |
| $[start:]$    | [start, 末尾] |
| $[:end]$      | [开头, end)   |

```py
s = "HELLO WORLD!"

# ELLO
s[1:6]

# LO WORLD!
s[3:]

# HELLO
s[:5]
```

### 函数

#### 分割

```py
split('字符')
```

```py
s = "a b c"

# ['a', 'b', 'c'] 
s.split()

# <class 'list'>
type(s.split())
```

### 查找

#### 子串查找

```py
find(s, start, end)
```

在字符串[start, end]范围里查找子串s, 若存在返回首个索引, 否则返回-1

```py
s = "abcdefg"

x = s.find("cd", 0, -1)

# 2
print(x)
```

#### 次数查找

```py
conut(s, start, end)
```

返回字符串[start, end]范围内s出现次数

### 过滤

#### 子符过滤

```py
strip(str)
```

过滤字符串开头与结尾中位于str中字符

```py
a = "a12312a"

# 12312
print(a.strip('a'))

# 2312
print(a.strip("a1"))

# 3
print(a.strip("12a"))
```
