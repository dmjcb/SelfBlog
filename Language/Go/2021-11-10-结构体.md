# Go结构体

## 定义

```go
type identifier struct {
    field1 type1
    field2 type2
    ...
}
```

## 初始化

### new

使用 new 函数给一个新的结构体变量分配内存, 它返回指向已分配内存的指针

```go
package main
import "fmt"

type Student struct {
    id int
    name string
    school string
}

func main() {
    stu := new(Student)
    stu.
}
```
