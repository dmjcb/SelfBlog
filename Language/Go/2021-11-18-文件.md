# Go文件

## 创建

```go
package main
import (
    "log"
    "os"
)

var (
    f *os.File
    err error
)

func main() {
    f, err = os.Create("文件名")
    if err != nil {
        log.Fatal(err)
    }
    log.Println(f)
    f.Close()
}
```
