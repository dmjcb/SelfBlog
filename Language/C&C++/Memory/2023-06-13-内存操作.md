# 内存操作

## sizeof

`sizeof` 是单目运算符

参数是数组、指针、类型、对象、函数等, 返回一个对象或者类型所占的`内存字节数`

| 参数类型              | 返回值                     | 举例                            |
| -------------------- | -------------------------- | ------------------------------ |
| 数组类型             | 数组的总字节                | int b[5]<br>sizeof(b)=20        |
| 具体的字符串或者数值  | 根据具体类型转化            | sizeof(8)= 4; 自动转化为int类型  |
| 指针                 | 64位系统: 8<br>32位系统: 4 |
