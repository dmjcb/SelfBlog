# 镜像指令

## 获取

### 拉取

```sh
docker pull [镜像名](版本, 默认latest)
```

- 拉取arm gcc镜像

```sh
docker pull arm32v7/gcc:9
```

## 操作

#### save

打包镜像

```sh
docker save -o [压缩包.jar] [镜像名/镜像ID]
```

#### load

解压镜像

```sh
docker load -i [压缩包]
```

#### tag

重命名镜像

```sh
docker tag 镜像ID 仓库:TAG
```

## 删除

#### 删除none镜像

```sh
docker rmi -f $(docker images | grep '<none>' | awk '{print $3}') 
```
